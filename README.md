# PHP keypad

## requirements
```php
0 => [' ', ',', 0]
1 => ['.', '!', '?', 1]
2 => ['a', 'b', 'c', 2]
3 => ['d', 'e', 'f', 3]
4 => ['g', 'h', 'i', 4]
5 => ['k', 'l', 'm', 5]
6 => ['n', 'o', 'p', 6]
7 => ['q', 'r', 's', 7]
8 => ['t', 'u', 'v', 8]
9 => ['w', 'x', 'y', 'z', 9]
'*' => break
'#' => toggle case
```

https://en.wikipedia.org/wiki/E.161
converts phone keypad input (based on the e.161 standard) to text
